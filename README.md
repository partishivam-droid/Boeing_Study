# Boeing Event Study: 2025 US-China Trade Tensions

This repository contains the replication package for an event study analyzing the impact of 2025 US-China trade tensions and policy announcements on Boeing (BA) cumulative abnormal stock returns.

## Overview
The analysis uses a two-step quantitative framework:
1. Data Ingestion & Event Study Windowing (Python): Pulls market data via Yahoo Finance, estimates a market model baseline using Calendar Year 2024 data, calculates Daily Cumulative Abnormal Returns (CAR) across six major 2025 geopolitical event windows, and exports a clean dataset.
2. Econometric Verification (Stata): Imports the structured time-series data to run a formal OLS regression with robust standard errors, evaluating the joint statistical significance of the trade shocks.


## File Structure

* `boeing_study.py`: Python script utilizing `yfinance` and `statsmodels` to download data, calculate baseline parameters, construct CAR trajectories, and plot visual outputs.
* `Boeing.do`: Stata do-file executing time-series setup (`tsset`), defining event window indicator variables, and running robust OLS regression analysis.
* `boeing_returns_data.csv`: The intermediate daily percentage returns dataset generated by Python and analyzed in Stata.
* `boeing_trade_war_study.png`: The finalized publication-grade line chart tracking CAR trajectories across the key event windows.


## Replication Steps

### Step 1: Python Data Processing
Run the Python script first to ingest raw market returns and generate the necessary datasets and figures:

    python boeing_study.py

*This will output the baseline Alpha/Beta calculations to your console, save the visual plot, and export `boeing_returns_data.csv`.*

### Step 2: Stata Econometric Analysis
Open Stata and run `Boeing.do`. The script will automatically:
1. Load the generated CSV file.
2. Format and `tsset` the daily time-series data.
3. Run the primary regression model with robust standard errors to output final coefficient tables and execute joint significance tests.

## Timeline of Evaluated 2025 Events
The study monitors a timeline of trade escalations and resolutions throughout 2025, tracking a window of -1 to +5 trading days around each announcement:
* April 2, 2025: Liberation Day reciprocal tariffs.
* April 9, 2025: Tariffs on Chinese goods raised to 145%.
* April 15, 2025: China orders Boeing delivery halt.
* May 12, 2025: Geneva trade truce announced (Boeing delivery ban lifted).
* October 10, 2025: Trump threatens 100% tariff after China rare earth export expansion.
* October 30, 2025: Busan Summit — US-China tariff and rare earth truce.
